Uses Grunt as build system. Therefore requires:
- Node.js
- npm
- a global installation of the grunt-cli		http://gruntjs.com/getting-started

It uses Composer as dependency manager for PHP.
- requires a global composer installation		https://getcomposer.org/doc/00-intro.md

It uses Sass and some libraries, Therefore it requires:
- Ruby and RubyGems					https://www.ruby-lang.org/en/documentation/installation/
- Sass							http://sass-lang.com/install
- breakpoint						https://github.com/at-import/breakpoint/wiki/Installation#installation
- susy							http://susydocs.oddbird.net/en/latest/install/#simple-install
- bourbon						(will be installed via npm install)




... got everything installed? Then do this, step by step:

- clone repository

- cd to the project and run
npm install
